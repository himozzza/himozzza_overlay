# Очень простой скрипт для обновления ebuild-файлов локального оверлея.

# Пример:
Скрипт получает версию пакета с GitHub и просто назначает версию ebuild, меняя название. От чего при синхронизацию мы видим обновление) 

Использовалось для zen-sources, так как обновление ядра zen для gentoo не очень торопится обновляться по умолчанию. 

В мапе gitLink указать путь до каталога с ebuild и адрес с releases с GitHub.

Просто утсанавливаем в автозапуск,радуемся. Желательно добавит в атозапуск с sudo, либо раздать права пользователю для локального оверлея.

Требуется заранее создать ebuild с версией из ветки gentoo. Сам он создаваться не будет.
